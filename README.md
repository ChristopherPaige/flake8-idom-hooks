# `flake8-idom-hooks`

A Flake8 plugin that enforces the rules of hooks for IDOM

# Install

```bash
pip install flake8-idom-hooks
```

# Developer Installation

```bash
pip install -r requirements.txt
pip install -e .
```

Run the tests

```bash
tox
```
